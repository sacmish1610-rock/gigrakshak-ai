## 🐛 If Something Goes Wrong

### Backend won't start?
```bash
cd backend
rm -r node_modules
npm install
npm run dev
```

### Frontend won't start?
```bash
cd frontend
npm cache clean --force
rm -r node_modules
npm install
npm run dev
```
